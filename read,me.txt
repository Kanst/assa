Скрипты для начального создания базы АССА
1_(Cre_main).sql - создает пустую базу данных
2_(data_start).sql - производит начальное заполнение нормативных данных
3_(channel).sql	- заполняет два канала данных для корректной работы ВПМ загрузки данных

Необходимо зайти в менеджер баз данных PostgresQL Pgadmin под именем пользователя assa
Создать новую базу данных или выбрать ранее созданную пустую базу данных
(в ней отсутствуют таблицы)
В меню pgAdmin  выбрать пункт "Редактор SQL" и зайти в редактор
В редакторе последовательно выполнить скрипты, начиная с 1-го 

в результате будет создана база данных асса, произведено начальное заполнение ее нормативными данными, 
установлена версия базы данных 

Текущая версия БД 1.2.4.0
